The Basics of the Web

The World Wide Web

The WWW is a collection of html documents. HTML is the basis for most every web page. Hyperlinks link the web together. Major Pieces of the web: You, Your computer and browser, The Internet, Servers, HTTP. HTTP (Hyper text transfer protocol) moves HTML files from servers to your browser.

HTML Documents

HTML documents are generally structured like this: 

<!DOCTYPE html>
<html>
<head>
	<title>Catchy Title</title>
</head>
<body>
	Here is some content!
</body>
</html>

Working with HTML and CSS

Creating a Structured Document
Congradulations, cadet. You've made it to CSS; much better place than Eden Prime. By now you should have learned that styling can be just as important as knowing the actual syntax. It's crucial that you learn to be consistent and organized when writing this code. Some of the concepts you should take note of are,
  • Avoid repetition: You can avoid having to repeat yourself in code and lower your risk at making mistakes by using language like CSS to control multiple tags and elements in HTML. This will make you more efficient, more acurate and make it easier for you to maintain your code over time.
  • Watch your indentation: keep everything clean and organized when writing script. Makes your life a lot easier down the road. When you read an HTML document as text, you see a wave of changing indentations going up and down the left side of the document. The more indented an element is, the more deeply nested its corresponding "box" is.
  • Be consistent: mind your letters and use of tags. For example, if you use lower case letters to define values or attributes in one area of your code, then make all your values and attribute lower case.
Boxes

By now you've also seen a box model. It's made up of content, padding, border and margin. By learning to manipulate the properties of this model, you'll be able to have a great impact on the user experience.

Everything on a web page is a box. This makes things easy to rearrange. To boxify a design, cut up the boxes physically or conceptually, and find ways to arrange them in a way that looks cool and user friendly.
